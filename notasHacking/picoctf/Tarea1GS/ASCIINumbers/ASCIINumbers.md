
# ASCII Numbers

## Descripción

Reto de decodificación de números ASCII en hexadecimal. Se proporciona una cadena de valores hexadecimales que representan caracteres ASCII. El objetivo es convertirlos a texto legible para obtener el flag.

## Solución

Valores dados:
```
0x70 0x69 0x63 0x6f 0x43 0x54 0x46 0x7b 0x34 0x35 0x63 0x31 0x31 0x5f 0x6e 0x30 0x5f 0x71 0x75 0x33 0x35 0x37 0x31 0x30 0x6e 0x35 0x5f 0x31 0x6c 0x6c 0x5f 0x74 0x33 0x31 0x31 0x5f 0x79 0x33 0x5f 0x6e 0x30 0x5f 0x6c 0x31 0x33 0x35 0x5f 0x34 0x34 0x35 0x64 0x34 0x31 0x38 0x30 0x7d
```

### Paso 1: Convertir los valores hexadecimales a texto

Puedes usar Python:
```python
nums = [0x70, 0x69, 0x63, 0x6f, 0x43, 0x54, 0x46, 0x7b, 0x34, 0x35, 0x63, 0x31, 0x31, 0x5f, 0x6e, 0x30, 0x5f, 0x71, 0x75, 0x33, 0x35, 0x37, 0x31, 0x30, 0x6e, 0x35, 0x5f, 0x31, 0x6c, 0x6c, 0x5f, 0x74, 0x33, 0x31, 0x31, 0x5f, 0x79, 0x33, 0x5f, 0x6e, 0x30, 0x5f, 0x6c, 0x31, 0x33, 0x35, 0x5f, 0x34, 0x34, 0x35, 0x64, 0x34, 0x31, 0x38, 0x30, 0x7d]
print(''.join(chr(n) for n in nums))
```
