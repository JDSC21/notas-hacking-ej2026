# Reto SafeOpener2

## Descripción 
What can you do with this file?
I forgot the key to my safe but this file is supposed to help me with retrieving the lost key. Can you help me unlock my safe?

## Solución

Instalar jadx decompilador de java
`sudo apt install jadx`

Ejecutar jadx-gui
`jadx-gui`

Abrir el archivo .class del reto :
 `File - Open Files`

Dentro se observa la bandera.
